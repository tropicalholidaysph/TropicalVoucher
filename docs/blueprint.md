# **App Name**: Tropical Ledger

## Core Features:

- Secure User Authentication: A secure login page allowing only authorized users to access and manage voucher data.
- Voucher Data Entry Form: A responsive form for digitizing manual voucher details including Voucher No, Date, Recipient, Amounts, Payment Method, Bank, Reference, and Purpose.
- Amount-to-Words Converter Tool: Automatically convert numerical 'Amount in R.O.' to its word format ('Sum of Rial Omani') for accurate representation on vouchers.
- Centralized Voucher Dashboard: A searchable and filterable table view, mirroring the provided Excel screenshot, displaying all historical vouchers with optional color-coding for status.
- Visual Voucher PDF Generator: Generate individual digital vouchers as PDFs, matching the layout and design of the physical 'Payment Voucher' with a placeholder for the company logo.
- Excel Data Export: Export all voucher data from the dashboard into a clean Excel (.xlsx) file, formatted consistently with the provided spreadsheet reference.
- Supabase Integration for Data Persistence: Store all voucher entries securely in a Supabase database, allowing users to access data history from any device.

## Style Guidelines:

- Color scheme: A professional yet inviting light theme. The primary hue, informed by the 'Tropical Holidays' concept and the warm tones of paper vouchers, is a soft, golden peach. Primary action color: #E66E38 (a warm, saturated peach). Background color: #F7F3F1 (a very light, desaturated peach, providing a clean canvas). Accent color: #DB0D3A (a deeper, rich red, providing contrast and urgency for key elements).
- Font: 'Inter' (sans-serif) for all text elements. Its modern, objective, and neutral aesthetic ensures high readability for data entry, tables, and reports across all screen sizes, aligning with a professional user experience.
- Utilize a consistent set of crisp, modern line-based icons for clarity. Icons should represent actions like 'Save', 'Export', 'Print', 'Search', and 'Edit', maintaining a professional and clean appearance.
- Implement a fully responsive design, optimized for both desktop and mobile viewing. The layout prioritizes clear organization of data entry forms, easy-to-read data tables with intuitive filtering and search, and clear call-to-action buttons for exports and PDF generation.
- Subtle and functional animations should enhance user feedback without being distracting. This includes smooth transitions for form submissions, data loading indicators, and highlight effects on interactive elements within the dashboard table.
